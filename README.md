# Random Quotes Generator

A simple and interactive web project that displays a new random quote every time the button is clicked. Built using HTML, CSS, and JavaScript.

## 🚀 Features
- Displays random inspirational quotes
- New quote generated on each button click
- Simple and clean user interface
- Beginner-friendly JavaScript logic

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
